# Sistema de Gestion de Comercio

Este sistema está pensado para gestionar un pequeño o mediano comercio.


## Principales funcionalidades:

### Usuarios
- Inicio de sesión mediante credenciales username y password

### Control de inventario
- Ingresar la mercadería al sistema cuando la trae el proveedor
- Consultar y gestionar información detallada de los productos
- Consultar y gestionar las categorías de productos
- Consultar y gestionar las marcas
- Consultar y gestionar los proveedores

### Ventas
- Registar ventas, con su respectivo detalle
- Consultar y gestionar los métodos de pago

### Dashboard
- Visualizar información sobre ventas y productos, con los principales indicadores tales como productos y fechas
